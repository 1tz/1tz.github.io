折腾树莓派

Posted by 1tz on January 6, 2018

抱歉，路由器已经满足不了我了

自从考完研就心水树莓派，于是马上就买了个折腾折腾，想当个软路由，但是效果并没有想象中的那么好。暂时先搁着，闲暇时候折腾折腾，当个小玩具。慢慢更新，免得以后迷路。

1. 准备工作

一根网线，一台电脑（只说 MAC OS 环境），一个 Raspberry Pi 3B，一个读卡器，一个8G及以上的 TF 卡，一个 Wi-Fi 环境，一个 USB 网线转换器

2. 钦定系统 Raspbian

2.1 烧录

下载官网的固件后，解压出 .img 格式的镜像文件后，插上读卡器。打开 iTerm / Terminal，输入以下指令查看当前挂载设备

df -h

找到你U盘的路径 [usb_device_path] 然后输入以下指令卸载设备

diskutil unmount [usb_device_path]

然后输入以下指令再找路径 [device_path]

diskutil list

然后就可以烧录了

dd ds=4m if=[path_to_img] of=[device_path]

等待10分钟左右即可，烧录完成后进入U盘路径，并创建空文件 ssh 后保存退出

cd [path_to_device]
touch ssh

2.2 连接

系统偏好设置->共享->互联网共享 把连接的Wi-Fi共享到USB端口，然后再打开 iTerm / Terminal 后输入以下指令

arp -a

就可以找到树莓派的IP地址，一般是 192.168.2.3 ，再使用SSH连接树莓派，密码是raspberry

ssh root@[ip_address]

至此即可使用 ssh 控制树莓派

3. 路由系统 LEDE

3.1 烧录

下载官网镜像，选择 arm_cortex-a53_neon-vfpv4 这个目录下面的固件，解压出 .img 格式的镜像文件后，插上读卡器。打开 iTerm / Terminal，输入以下指令查看当前挂载设备

df -h

找到你U盘的路径 [usb_device_path] 然后输入以下指令卸载设备

diskutil unmount [usb_device_path]

然后输入以下指令再找路径 [device_path]

diskutil list

然后就可以烧录了

dd ds=2m if=[path_to_img] of=[device_path]

等待5分钟左右即可

3.2 连接

现在官方的 LEDE 固件已经集成 LUCI ，并开启了 DHCP 。网线连到树莓派的网线口，关掉Wi-Fi，有IP地址后，登录 192.168.1.1 即可进行路由器设置

3.3 设置

打开 LUCI 界面，开启树莓派的无线，然后用 Wi-Fi 连接树莓派。（未完待续）
